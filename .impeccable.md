# Impeccable — Design Context

## Users
- Operators running a ROI-driven marketing studio who manage this dashboard daily to create case studies, track campaign results, and close new clients in live meetings.
- Context: quiet daily ops inside an office, and high-stakes live demos inside prospect meetings. Readability from a laptop screen + projection must both work.

## Brand Personality
Three words: **crédible, tranchant, rentable**.
Voice: factual, confident, no-fluff. Results over narrative. Currency and ROI numbers are the heroes.

## Aesthetic Direction
- **SaaS premium** inspired by Linear, Vercel, Stripe and Notion, not "marketing page with gradients".
- **Theme**: light. Professional office use, calm surfaces, data visibility in daylight.
- **Palette**: brand `#5694bd` / `#3e6493` / `#2a2e5e` used sparingly as accent. 60% neutral whites / tinted grays / 30% text & borders / 10% brand.
- **Type pairing**: Inter for UI body (project mandate), paired with **Archivo** (400 / 500 / 600 / 800 / 900) for display headings and hero ROI numbers. Tabular numbers everywhere currency / counts appear.
- **Space**: generous whitespace, 4pt rhythm, asymmetric hero composition, tight tables, left-aligned headers.
- **Anti-references**: no gradient text, no left-border accent stripes, no identical card grids, no decorative sparklines, no neon-on-dark, no rounded-corner logo blocks above every heading.

## Design Principles
1. **Numbers first.** ROI, CA, ROAS are the emotional payload — they should be the largest visible element on any screen they appear on.
2. **Quiet chrome, loud data.** Neutral surfaces and borders; accent color reserved for active state, primary CTA, and KPI pops.
3. **Progressive density.** Dashboards use spacious cards; lists tighten up to maximize scan. Forms break the grid with section rails.
4. **Every interaction <300ms.** Motion for state changes only (entrance / exit / toggle). No bounce, no parallax, no spring tackiness.
5. **Live-demo ready.** Sticky save bars, keyboard-friendly, never require a scroll to see the headline KPI.
